# A.C.T. San Francisco · 2025–26 Season Microsite

Single-page microsite for American Conservatory Theater featuring the 2025–26 season.

## Features

- **Fully responsive** mobile-first design
- **Single HTML file** - no build process required
- **Smooth animations** and scroll behavior
- **Accessible** and semantic HTML
- **Production-ready** for immediate deployment

## Design

- **Colors**: Near-black background (#0a0a0a), off-white text (#f5f0e8), gold accent (#c9a84c)
- **Typography**: Playfair Display (headings) + Inter (body) via Google Fonts
- **Layout**: Hero section, season cards, about block, conservatory callout, footer

## Deployment

This site is designed to deploy instantly to:
- **Vercel** - Zero configuration
- **Netlify** - Drag and drop
- **GitHub Pages** - Enable in repo settings

## Local Development

Simply open `index.html` in your browser. No build tools required.

## Content Updates

All content is in the single `index.html` file. Update text, links, and styling directly in the HTML.
